# Amazon
This project is a practice of maven project with implementation of TestNG and POM using pagefactory.
This project opens up website in firefox using geckodriver.
This project has three basic tests senarios to execute.
1. It trys to Login in the web page by sending one positive and one negative credentiontials and returns to home page.
2. It trys to create new account and passes some positive and some negative credentials to reach submit button and returns to home page.
3. It trys to input some texts in search input feild and clicks on search button then scrolls down to a point and returns to homepage.
